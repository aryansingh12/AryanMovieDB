# Android Application
Complete movie/tv show info app with in-app trailers

## Technology used
<ul>
<li>The Movie Database (TMDb)</li>
<li>SQLite Database</li>
<li>Retrofit - for networking</li>
<li>BroadcastReceiver</li>
</ul>
